# Hostel Management Mobile Application

This is a full-stack mobile application developed for managing hostel operations such as student management, room allocation, and financial tracking.

The system consists of a backend (Node.js + Express + MongoDB) and a mobile frontend built using React Native (Expo).


##  Technologies Used

Frontend:
- React Native (Expo)
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

Tools:
- Android Studio (Emulator)
- VS Code
- Git & GitHub
